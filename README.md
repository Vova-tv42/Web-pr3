# Інструкція запуску

Для цього проекту використовується PostgreSQL. Перед запуском переконайтесь, що [PostgreSQL](https://www.postgresql.org/download/) встановлений і утиліта psql доступна в терміналі.

1. **Встановіть пакети:**

  ```
  npm install
  ```

2. Налаштуйте параметри підключення до бази даних у файлі server.js (змінна pool) відповідно до ваших актуальних значень.

3. **Ініціалізуйте базу даних:**

  ```
  npm run db:init
  ```

4. **Запустіть застосунок:**

  ```
  npm run start
  ```

5. Відкрийте у браузері: http://localhost:3000/
